# Omara Payments Design System

This document outlines the core design principles, color palette, and typography used across the Omara Payments platform.

## 1. Typography

We use **Geist Sans** (implemented via Inter as a visual alternative) for a clean, modern, and highly readable interface suitable for financial data.

- **Font Family**: `'Inter', sans-serif`
- **Variable**: `--font-geist`
- **Weights**: 
  - Regular (400) - Body text
  - Medium (500) - Labels, Buttons
  - Semibold (600) - Headings, Data points
  - Bold (700) - Major Headings

## 2. Color Palette

The color system is built using CSS variables to support dark mode and theming.

### Primary Colors
- **Primary (Deep Midnight Blue)**: `#0F172A` (Tailwind: `slate-900`)
  - Used for: Main navigation, primary text, dark mode backgrounds.
- **Accent (Emerald Green)**: `#10B981` (Tailwind: `emerald-500`)
  - Used for: Success states, positive financial data, call-to-actions.

### UI Colors
- **Background Light**: `#F9FAFB` (Tailwind: `slate-50`)
- **Background Dark**: `#020617` (Tailwind: `slate-950`)
- **Border Light**: `#E2E8F0`
- **Border Dark**: `#334155`

### Special Effects
- **AI Gradient**: `linear-gradient(135deg, #8B5CF6 0%, #6366F1 100%)`
  - Used for: AI features, premium badges, highlighted text.

## 3. Glassmorphism

Glass effects are used to create depth and a modern fintech feel.

- **Class**: `.glass-panel`
- **Properties**:
  - Backdrop Blur: `12px`
  - Background: `rgba(255, 255, 255, 0.7)` (Light) / `rgba(15, 23, 42, 0.6)` (Dark)
  - Border: `1px solid rgba(255, 255, 255, 0.5)`

## 4. Components

### Buttons
- **Touch Target**: Minimum 44x44px on mobile devices.
- **Padding**: `px-4 py-2` (Desktop), `px-4 py-3` (Mobile).
- **Radius**: `rounded-xl` or `rounded-lg`.

### Cards
- **Base Style**: White background, subtle shadow, rounded-2xl borders.
- **Interactive**: Hover lift (`-translate-y-1`), shadow increase.

## 5. Mobile Responsiveness

- **Breakpoints**: 
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px
- **Tables**: Converted to stacked cards on mobile views (< 768px).
- **Navigation**: Sidebar collapses to a Drawer (Sheet) on mobile.

## 6. Implementation Guide

To use colors in Tailwind:
- `bg-brand-primary`
- `text-brand-accent`
- `bg-ai-gradient`

To use the glass effect:
- Add `glass-panel` class to any `div` or `Card`.
